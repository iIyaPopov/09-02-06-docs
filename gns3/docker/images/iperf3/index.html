    iperf3

    Установка и настройка

    Для тестирования скорости соединения необходимы сервер и клиент.

    Сервер (С)

    Чтобы сделать из шаблона С, необходимо разместить устройство на схеме и открыть его конфигурацию (ПКМ > Configure). В появившемся окне в поле Start command необходимо вписать

    -s

    Клиент (К)

    Чтобы сделать из шаблона К, необходимо разместить устройство на схеме и открыть его конфигурацию (ПКМ > Configure). В появившемся окне в поле Start command необходимо вписать

    -с IP-адрес сервера

    Использование iperf3

    Внимание

    Данное задание выполняется в рамках МДК.01.02 (часть 3).

    Что значит "Выполнить тест"?

    Далее под выполнением теста подразумевается запуск клиента для тестирования скорости соединения. Сервер при этом должен быть уже запущен. После выполнения тестирования К автоматически отключается. Через терминал сервера необходимо посмотреть статистику.

    Примечание

    Для тестов ниже может понадобиться изменить IP-адреса. Делать это удобнее через меню ПКМ > Edit config. Каждый тест выполнять независимо от других.

    Тесты

    Тест 1
    • Соединить С и К напрямую.
    • Выполнить тест.
    Тест 2
    • Соединить С и К через простой нативный коммутатор (не Cisco и не MikroTik).
    • Выполнить тест.
    Тест 3
    • Соединить С и К через коммутатор Cisco.
    • Выполнить тест.
    Тест 4
    • Соединить С и К через два коммутатора Cisco (все устройства должны быть соединены последовательно).
    • Выполнить тест.
    Тест 5
    • Соединить С и К через маршрутизатор Cisco.
    • Выполнить тест.
    Тест 6
    • Настроить на маршрутизаторе Cisco ограничение скорости (как это сделать, можно посмотреть здесь) до нескольких Кб/Мб в секунду.
    • Выполнить тест.
