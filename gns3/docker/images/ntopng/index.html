    Использование docker-образа ntopng.

    Перед его использованием необходимо скачать или импортировать docker-образ на GNS3 VM. Подробнее об этом описано в разделе о работе с docker-образами.

    Настройка шаблона устройства в GNS3.

    Откройте свойства шаблона. В поле start command написать

    --dns-mode 2 --interface eth0

    Подробнее о данных параметрах можно посмотреть в документации ntop.

    В поле HTTP port in the container указать 3000. Можно указать другое значение, но тогда необходимо менять значенния в других полях.

    Настройка IP-адреса

    Адрес можно назначить вручную через config (ПКМ > Edit config). Для этого необходимо раскоменитировать строки и поменять IP-адрес

    auto eth0
iface eth0 inet static
    address 192.168.126.145
    netmask 255.255.255.0

    В этом случае IP-адрес известен и по нему можно открыть веб-консоль ntopng.

    Чтобы получить адрес автоматически, необходимо раскоменитировать строки

    auto eth0
iface eth0 inet dhcp

    В этом случае IP-адрес сразу неизвестен, поэтому необходимо сначала открыть Console у ntopng, и через нее выяснить IP-адрес (данный способ может быть сложнее при наличии большого количества устройств на схеме). В Configure шаблона в поле Console type необходимо указать http.

    Примечание. При открытии Console могут наблюдаться задержки.

    Открытие веб-консоли ntopng.

    Чтобы открыть веб-консоль, необходимо в браузере написать :<3000>. Если ранее был указан не 3000 порт, то здесь нужно написаать указанный ранее порт.

    Адрес указывается, который был назначен ранее.
